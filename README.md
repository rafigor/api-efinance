# eFinance API

Projeto eFinance API

## Install the dependencies
```bash
composer install
```
## Configure .env
```bash
cp .env.example .env
```

### Start the app in development mode (hot-code reloading, error reporting, etc.)
```bash
php -S localhost:8000 -t public
```
